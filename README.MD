## ğŸ¨ Design & Dev Challenges

Being a **Designer transitioning to Front-end**, this project was a significant milestone:

* **CSS Compatibility:** Handled and solved issues with modern CSS Nesting on older browsers (like iOS 15/iPhone SE), refactoring to a robust "Flat CSS" structure.

* **Sequential Delays:** Implementing precise `setTimeout` and `animationDelay` logic to ensure a smooth reveal of results.
